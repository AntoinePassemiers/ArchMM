class archmm.hmm.HMM(n_states, arch='ergodic')¶

Bases: object

Base class for Hidden Markov Models

Parameters:
  • n_states (int) – Number of hidden states
  • arch (str) – Topology name: can be either ‘ergodic’, ‘linear’ or ‘cyclic’
n_features¶

int – Dimensionality of the observed samples

ln_initial_probs¶

np.ndarray – Logarithm of state start probabilities, where ln_initial_probs[i] is the probability that the model starts in state i when generating a new sequence

ln_transition_probs¶

np.ndarray – Logarithm of state transition probabilities, where ln_transition_probs[i, j] is the probability that the model switches from state i to state j

initial_probs¶

np.ndarray – State start probabilities

transition_probs¶

np.ndarray – State transition probabilities

class archmm.hmm.MHMM(n_states, arch='ergodic')¶

Bases: archmm.hmm.HMM

Multinomial Hidden Markov Model

n_unique¶

int – Number of unique values that can be observed

proba¶

np.ndarray – Array of shape (n_states, n_unique) where proba[i, j] is the probability of observing value j when being in hidden state i.

emission_log_proba(self, X)¶

Computes emission log-probabilities of a given sequence according to the parameters of the multinomial distributions of each state.

Parameters:X (np.ndarray) – Sequence of observed samples
Returns:Emission log-probabilities of given sequence
Return type:np.ndarray
estimate_params(self, X_s)¶

Pre-estimates model parameters

Parameters:X_s (list) – List of sequences, where sequence i is a NumPy array of shape (n_samples_i, n_features)
get_num_params_per_state(self)¶

Returns an upper bound on the number of free parameters for a single hidden state.

The number of required parameters is equal to n_unique - 1 since any emission probability vector sums to one.

Returns:Upper bound on the number of free parameters per state
Return type:int
